# Задание 1



## Задача 1



### Текст задачи



Получить список из произведений цифр соответствующих чисел, хранящихся в исходном списке. 



### Алгоритм решения



1. Запрашиваем количество элементов списка у пользователя.
2. Создаем список с заданным количеством элементов.
3. С помощью List.map (применение функции к каждому элементу, возвращение измененного списка) каждый элемент:

&nbsp;   1. Проверяем на частный случай (0), возвращаем 0.

&nbsp;   2. Иначе вычисляем произведение цифр в элементе и возвращаем полученный результат, как элемент измененного списка.

4\. Выводим изменненый список.



### Тестирование



Скриншоты результата работы программы



# Задание 2



## Задача 1



### Текст задачи



Список содержит строки, обозначающие двоичные числа от 1 до 9. Найти в десятичной системе счисления сумму элементов списка.



### Алгоритм решения



1. Запрашиваем количество элементов списка у пользователя.
2. При создании списка:

&nbsp;   1. Проверяем, что пользователь вводит бинарные значения.

&nbsp;   2. Проверяем, что бинарное значение в диапазоне от 1 до 9.

3\. При вычислении десятеричной суммы бинарных значений:

&nbsp;   1. Преобразуем бинарное значение в десятеричное (число).

&nbsp;   2. С помощью List.fold (сведение к одному значению) сводим все полученные числа к одному значению (сумме).

4\. Выводим полученную сумму.



### Тестирование



Скриншоты результата работы программы
